# 채팅 인터페이스

[!DNL Adobe Journey Optimizer B2B Prime] 전체에 채팅 패널이 포함되어 있습니다. 일반 언어를 사용하여 AI 에이전트와 상호 작용하여 프로그램 및 여정을 빌드하고, 사용자 목록을 만들고 비교하며, 리드를 조사하고, 채점을 구성하는 등의 작업을 수행합니다.

왼쪽 탐색에서 _AI Assistant_ 아이콘을 선택하여 패널을 엽니다. 패널 헤더에는 네 개의 컨트롤이 있습니다.

| 컨트롤 | 설명 |
|---------|-------------|
| **새 대화** | 새 채팅을 시작합니다(현재 스레드가 지워짐). |
| **대화 기록** | 지난 대화를 열어 다시 열 수 있습니다. |
| **패널 교체** | 채팅 패널을 작업 영역의 다른 쪽으로 이동합니다. |
| **축소** | 작업 영역에 더 많은 공간을 제공하려면 패널을 숨깁니다. |

패널 하단에 메시지 상자를 열어 다음과 같은 작업을 수행할 수 있습니다.

* 메시지를 추가하고 **Enter**&#x200B;을 눌러 보내세요(**Shift+Enter**&#x200B;이(가) 새 행을 삽입합니다).
* _첨부_ 아이콘을 사용하여 파일을 첨부합니다(지원되는 형식: `.txt`, `.md`, `.csv`, `.json`, `.xlsx`, `.docx`, `.pdf`). CSV 및 스프레드시트 업로드를 사용하여 리드 가져오기를 시작합니다.

## AI 지원에 문의

작업을 완료하는 두 가지 유효한 방법이 있습니다. 슬래시 메뉴를 사용할 필요가 없습니다.

**자연어** — 동료에게 보내는 방식으로 요청을 입력합니다.

* _&quot;새 평가판 등록에 대한 시작 여정을 만듭니다.&quot;_
* _&quot;john@acme.com에서 이 여정을 입력하지 않은 이유는 무엇입니까?&quot;_
* _&quot;내 &#39;Q3 웨비나 참석자&#39;와 &#39;엔터프라이즈 계정&#39; 목록을 비교합니다.&quot;_

상담원은 사용자의 말을 뒤에서 적절한 기술과 일치시키고 적절한 워크플로우를 실행합니다.

**슬래시(/) 메뉴** — `/`을(를) 입력하여 도우미가 수행할 수 있는 모든 작업을 검색할 수 있는 메뉴를 엽니다. 이 기능은 기능을 검색하거나 알려진 워크플로우로 바로 이동하려는 경우에 유용합니다.

## 슬래시(/) 메뉴

메시지 상자의 시작 부분이나 공백 뒤에 `/`을(를) 사용하여 메뉴를 엽니다. 입력을 계속하면 이름, 설명 또는 ID별로 목록 필터가 표시됩니다. 예를 들어 `/journey`은(는) 여정 관련 명령으로 좁혀집니다.

항목을 이동하려면 **/**&#x200B;을(를) 사용하고, 선택하려면 **Enter** 또는 **Tab**&#x200B;을(를) 사용하고, 취소하려면 **Esc**&#x200B;을(를) 사용하세요. 항목을 직접 클릭할 수도 있습니다.

메뉴 항목은 레이블이 지정된 섹션으로 그룹화됩니다.

| 섹션 | 다음 포함 |
|---------|----------|
| **조사** | 진단, 읽기 전용 조회(예: 잠재 고객 조사, 의도 쿼리). |
| **유효성 검사** | QA 및 감사 워크플로우. |
| **빌드** | 만들기 워크플로(프로그램, 여정, 목록, 점수). |
| **가져오기** | CSV 리드 가져오기. |
| **기타** | 위의 범주에 포함되지 않는 모든 항목. |

또한 메뉴에는 작업 영역 화면으로 이동하는 **커넥터**(예: _Marketo 찾아보기_&#x200B;에서 선택기 대화 상자가 열림)와 **탐색 바로 가기**&#x200B;가 나열됩니다.

### 메뉴 항목 선택

메뉴에서 스킬 또는 에이전트를 선택하면 편집할 수 있도록 시작 메시지가 메시지 상자에 삽입됩니다. **자동으로 전송되지 않음**&#x200B;입니다. 예를 들어 _캠페인 계획_&#x200B;을 선택하면 _에 삭제됩니다. [캠페인 이름]에 대한 새 Marketo 프로그램 계획 개요를 업로드하겠습니다.&quot;_ 대괄호로 묶인 자리 표시자를 입력한 다음 **Enter**&#x200B;을 눌러 보냅니다.

커넥터는 텍스트를 삽입하는 대신 모달을 엽니다. 탐색 단축키를 사용하면 작업 공간의 해당 화면으로 바로 이동할 수 있습니다.

>[!NOTE]
>
>일부 명령이 회색으로 표시되고 _준비 중_(으)로 표시됩니다. 기능 플래그로 제어되며 계정에서 아직 활성화되지 않았습니다. 기능 플래그를 선택하면 아무 작업도 수행되지 않습니다. 사용 가능한 세트는 사용할 수 있는 기능에 따라 다릅니다.

## 스킬

스킬은 에이전트가 실행 방법을 알고 있는 패키지화된 워크플로우로서, `/` 메뉴와 자연어 요청 모두의 뒤에 기본 요소가 있습니다. 각 스킬은 단계별 지침 및 한 작업에 필요한 특정 도구(예: &quot;여정 게시&quot;, &quot;두 사람 목록 비교&quot;, &quot;채점 모델 구축&quot;)를 번들로 제공합니다.

기술에 대해 알아야 할 주요 사항:

* **스킬은 제품 범위입니다.** AJO B2B Prime에서 AJO B2B 스킬(여정, 사람 목록, 점수, 채널, 전송 시간 최적화 등)을 볼 수 있습니다. 몇 가지 기술은 Marketo 전용이며 두 제품(리드 가져오기, 제품 지식)에서 몇 가지 작업이 가능합니다. 현재 위치와 관련된 스킬만 표시됩니다.
* **기술 이름을 기억할 필요가 없습니다.** 목표를 설명하고 에이전트가 일치하는 스킬을 선택합니다. `/` 메뉴는 동일한 워크플로우에 대한 검색 가능한 빠른 단축키입니다.
* **일부 스킬은 읽기만 하고 다른 스킬은 변경합니다.** 조사 및 보고 기술(예: 리드 조사, 의도 쿼리, 전송 시간 보고서)은 데이터만 읽습니다. 스킬 구축 및 구성(예: 여정 생성, 채점) 데이터 생성 또는 변경.

## 후속 프롬프트

길잡이가 답글을 보내면 수행한 작업에 맞게 클릭 가능한 후속 프롬프트가 한 줄로 표시되는 경우가 많습니다. 예를 들어 여정을 만든 후 _&quot;이 여정을 게시합니다.&quot;_ 또는 _&quot;대기 단계를 추가합니다.&quot;_. 입력하지 않고 계속하려면 하나를 클릭하십시오. 이는 제안일 뿐입니다. 대신 언제든지 다음 메시지를 입력할 수 있습니다.

## 팁

* **식별자를 사용하여 지정합니다.** 조사하거나 편집할 때 에이전트가 올바른 에셋에 대해 작동하도록 이메일 주소, 개인 이름, 여정 이름 또는 목록 이름을 포함합니다.
* **탐색하려면 `/`을(를) 사용합니다.** 길잡이가 수행할 수 있는 작업을 모를 경우 `/` 메뉴를 열고 범주를 선택하십시오.
* **시작 메시지를 편집합니다.** 스킬을 선택하면 템플릿이 제공됩니다. 보내기 전에 `[bracketed]` 자리 표시자를 바꾸십시오.
* **가져오기를 위해 먼저 업로드하십시오.** 잠재 고객 가져오기의 경우 먼저 CSV를 첨부한 다음 수행할 작업을 설명합니다.
* 관련 없는 작업으로 전환하면 **새 대화를 시작**&#x200B;합니다. 이전 컨텍스트는 새 요청에 영향을 주지 않습니다.
