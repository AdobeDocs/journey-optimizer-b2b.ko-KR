# 개인 여정 노드

[개인 여정](./person-journeys.md)을(를) 만든 후 캔버스에 있는 도구를 사용하여 대상자를 추가하고 노드를 사용하여 여정을 빌드합니다. 여정 캔버스에서는 여러 단계의 B2B 마케팅 사용 사례를 빌드할 수 있는 시각적 디자인 공간을 제공합니다.

_[!UICONTROL 개인 대상]_ 노드는 자동으로 여정의 첫 번째 노드입니다. 대상을 선택한 후 다양한 작업, 이벤트 및 의사 결정 노드를 다중 단계 크로스 채널 시나리오로 결합하여 여정을 빌드합니다. 여정의 각 노드는 논리적 경로를 따라 단계를 나타냅니다.

:::: landing-cards-container

:::
<!-- ![Person audience node](https://cdn.experienceleague.adobe.com/icons/users.svg) -->

개인 대상 노드

여정은 항상 개인 대상 노드로 시작합니다. 정적 또는 동적 사람 목록을 사용하여 입력 대상을 설정하여 여정에 입력하는 사람 프로필을 정의합니다.

[개인 대상 노드에 대해 알아보기](./person-audience-node.md)
:::

작업 노드

이메일 보내기 또는 데이터 값 속성 업데이트와 같이 경로의 모든 사용자에게 변경 사항을 적용합니다.

[작업 노드에 대해 알아보기](./action-nodes.md)
:::

:::
<!-- ![Event nodes](https://cdn.experienceleague.adobe.com/icons/puzzle-piece.svg) -->

이벤트 노드

이메일 열기 또는 데이터 값 변경과 같은 특정 활동 또는 데이터 변경이 발생할 때 사람들을 다음 여정 단계로 안내합니다.

[이벤트 노드에 대해 알아보기](./listen-for-event-nodes.md)
:::

대기 노드

사람들을 다음 단계로 이동하기 전에 설정된 기간 동안 또는 특정 날짜 및 시간까지 여정 진행률을 일시 중지합니다.

[대기 노드에 대해 알아보기](./wait-nodes.md)
:::

경로 노드 분할 및 병합

[분할 및 병합 경로 노드에 대해 알아보기](./split-merge-paths-nodes.md)
