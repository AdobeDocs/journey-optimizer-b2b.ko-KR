# 랜딩 페이지 템플릿

디자인 프로세스를 가속화하고 개선하기 위해 독립형 랜딩 페이지 템플릿을 만들어 페이지 디자인과 콘텐츠를 표준화할 수 있습니다. 마케팅 전략은 페이지를 재사용하고 캠페인 및 여정에 맞게 조정할 수 있습니다.

## 랜딩 페이지 템플릿 액세스 및 관리

Adobe Journey Optimizer B2B edition의 랜딩 페이지 템플릿에 액세스하려면 **[!UICONTROL 콘텐츠 관리]** > **[!UICONTROL 템플릿]**(으)로 이동하십시오. 탐색 패널의 _[!UICONTROL WEB]_&#x200B;에서 **[!UICONTROL 랜딩 페이지 템플릿]**&#x200B;을 선택합니다.

표시된 목록 페이지에는 테이블 형식으로 나열된 인스턴스에서 만든 모든 랜딩 페이지 템플릿이 포함됩니다. 표는 기본적으로 _[!UICONTROL 수정됨]_ 열을 기준으로 정렬되며 가장 최근에 업데이트된 템플릿이 맨 위에 있습니다. 오름차순과 내림차순 간을 변경하려면 열 제목을 클릭합니다.

이름별로 템플릿을 검색하려면 검색 막대에 텍스트 문자열을 입력합니다.

![랜딩 페이지 템플릿 라이브러리에 액세스하고 이름별로 필터링](./assets/landing-page-templates-list-search.png){width="700" zoomable="yes"}

왼쪽 상단의 _필터_ 아이콘( ![필터 아이콘](../assets/do-not-localize/icon-filter.svg) )을 클릭하여 생성 또는 수정 날짜 및 생성 또는 수정한 템플릿에 따라 목록을 필터링합니다.

오른쪽 상단의 _표 사용자 지정_ 아이콘(![표 사용자 지정 아이콘](../assets/do-not-localize/icon-column-settings.svg))을 클릭하여 표에 표시할 열을 사용자 지정합니다. 표시할 열을 선택하고 **[!UICONTROL 적용]**&#x200B;을 클릭합니다.

표시된 템플릿 목록에서 다음 섹션에 설명된 작업을 수행할 수 있습니다.

## 랜딩 페이지 템플릿 만들기

오른쪽 상단의 **[!UICONTROL 템플릿 만들기]**&#x200B;를 클릭하여 랜딩 페이지 템플릿 목록 페이지에서 랜딩 페이지 템플릿을 만들 수 있습니다.

1. 대화 상자에서 고유한 **[!UICONTROL 이름]**(필수)과 유용한 **[!UICONTROL 설명]**(선택 사항)을 입력하십시오.

   ![새 랜딩 페이지 템플릿의 초기 속성을 입력하십시오](./assets/landing-page-templates-create-dialog.png){width="400"}

1. **[!UICONTROL 만들기]**&#x200B;를 클릭합니다.

_[!UICONTROL 기본 랜딩 페이지 만들기]_ 페이지가 열리고 템플릿 만들기 옵션을 제공합니다. _[!UICONTROL 처음부터 디자인]_, _[!UICONTROL HTML 가져오기]_ 또는 _[!UICONTROL 저장된 템플릿 중 하나]_.

![랜딩 페이지 템플릿 디자인 시작 방법을 선택하세요](./assets/landing-page-templates-create-design.png){width="800" zoomable="yes"}

템플릿 디자인을 시작하는 데 사용할 메서드를 선택한 후 시각적 디자인 공간을 사용하여 [랜딩 페이지 템플릿 콘텐츠를 디자인합니다](./landing-page-design.md).

### 처음부터 디자인

시각적 디자인 공간을 사용하여 랜딩 페이지 콘텐츠의 구조를 정의합니다. 간단한 드래그 앤 드롭 작업으로 구조 구성 요소를 추가 및 이동하여 초 내에 재사용 가능한 페이지 콘텐츠의 형태를 디자인할 수 있습니다.

>[!NOTE]
>
>사용 가능한 디자인 도구는 랜딩 페이지 디자인에 사용되는 도구와 동일합니다. 차이점은 이 콘텐츠가 여러 랜딩 페이지에서 재사용할 수 있는 템플릿으로 저장된다는 것입니다.

1. _[!UICONTROL 템플릿 디자인]_ 홈 페이지에서 **[!UICONTROL 처음부터 디자인]** 옵션을 선택합니다.

1. 템플릿에 [구조 및 콘텐츠 추가](./landing-page-design.md#add-structure-and-content).

### HTML 가져오기

Adobe Journey Optimizer B2B edition을 사용하면 기존 HTML 콘텐츠를 가져와서 랜딩 페이지 템플릿을 디자인할 수 있습니다.

{{$include /help/_includes/content-design-import.md}}

![zip 파일에서 html 콘텐츠 가져오기](./assets/templates-import-zip-file.png){width="500"}

필요에 따라 시각적 디자인 공간을 사용하여 가져온 콘텐츠를 개인화할 수 있습니다.

## 페이지 템플릿 세부 정보 보기

_랜딩 페이지 템플릿_ 목록 페이지에서 랜딩 페이지 템플릿의 이름을 클릭하여 세부 정보 페이지를 엽니다. 여기에서 랜딩 페이지 템플릿에 대한 기본 속성을 보고, 시각적 디자인 공간에 액세스하여 템플릿 콘텐츠를 변경할 수 있습니다.

![랜딩 페이지 템플릿 세부 정보 검토](./assets/template-details.png){width="700" zoomable="yes"}

* 이름, 설명 등 템플릿 세부 정보를 봅니다. 이러한 설정은 편집할 수 있습니다. 변경 사항을 자동으로 저장하려면 설명 상자 외부를 클릭합니다.

* 작성자, 생성자, 마지막 업데이트 일자 및 수정자 등의 템플릿 속성을 확인합니다.

* 랜딩 페이지 템플릿에서 _복제_ 및 _삭제_&#x200B;와 같은 빠른 작업을 수행하려면 오른쪽 상단의 **[!UICONTROL 자세히]**&#x200B;를 클릭하십시오.

* 활성 알림(랜딩 페이지 템플릿에 대한 오류 및 경고)이 있는 경우, 오른쪽 상단의 **[!UICONTROL 알림]**&#x200B;을 클릭하여 정보를 확인하십시오.

  이러한 경고는 랜딩 페이지 생성을 위한 랜딩 페이지 템플릿 사용을 금지하지 않습니다. 이 정보는 팀원의 마케터에게 작동하지 않을 수 있는 내용과 이 정보를 게재에 사용하기 전에 필요한 업데이트에 대한 가시성을 제공합니다.

## 참조에서 사용하는 템플릿 보기

템플릿 세부 정보 페이지에서 **[!UICONTROL 사용한 사람]** 탭을 클릭하여 랜딩 페이지에서 이 템플릿이 사용되는 위치에 대한 세부 정보를 봅니다.

![템플릿 사용량을 확인하려면 [사용] 탭을 클릭하세요](./assets/template-details-used-by.png){width="400"}

* 링크를 클릭하면 템플릿이 사용되는 해당 랜딩 페이지로 이동합니다.

* 뒤로 화살표를 클릭하여 언제든지 보기를 종료하고 목록 페이지로 돌아갑니다.

## 랜딩 페이지 템플릿 편집

이 작업은 다음 위치에서 수행할 수 있습니다.

* 세부 정보 페이지 - **[!UICONTROL 랜딩 페이지 템플릿 편집]**&#x200B;을 클릭합니다.
* 목록 페이지 - 템플릿 옆에 있는 생략 부호(**...**)를 클릭하고 **[!UICONTROL 편집]**&#x200B;을 선택합니다.

이 작업을 수행하면 랜딩 페이지 템플릿의 마지막 저장 상태에 따라 _템플릿 디자인_ 페이지 또는 시각적 콘텐츠 편집기 페이지로 이동합니다. 여기에서 필요에 따라 랜딩 페이지 템플릿 콘텐츠를 편집할 수 있습니다. 편집 옵션에 대한 자세한 내용은 [랜딩 페이지 템플릿 만들기](#create-a-landing-page-template)를 참조하십시오.

## 랜딩 페이지 템플릿 복제

다음 방법 중 하나를 사용하여 랜딩 페이지 템플릿을 복제할 수 있습니다.

* 오른쪽의 템플릿 세부 정보에서 **[!UICONTROL 자세히]**&#x200B;를 확장하고 **[!UICONTROL 복제]**&#x200B;를 클릭합니다.

  ![삭제 및 복제 작업에 액세스하려면 자세히 클릭](./assets/template-details-more-menu.png){width="400"}

* _[!UICONTROL 랜딩 페이지 템플릿]_ 목록 페이지에서 템플릿 옆에 있는 생략 부호(...)를 클릭하고 **[!UICONTROL 복제]**&#x200B;를 선택합니다.

대화 상자에서 유용한 이름(고유)과 설명을 입력합니다. 작업을 완료하려면 **[!UICONTROL 복제]**&#x200B;를 클릭하십시오.

그러면 복제된(새) 랜딩 페이지 템플릿이 _랜딩 페이지 템플릿_ 목록에 나타납니다.

## 랜딩 페이지 템플릿 삭제

랜딩 페이지 템플릿 제거는 실행 취소할 수 없으므로 삭제 작업을 시작하기 전에 확인하십시오. 다음 방법 중 하나를 사용하여 랜딩 페이지 템플릿을 삭제할 수 있습니다.

* 오른쪽의 템플릿 세부 정보에서 **[!UICONTROL 자세히]**&#x200B;를 확장하고 **[!UICONTROL 삭제]**&#x200B;를 클릭합니다.
* _랜딩 페이지 템플릿_ 목록 페이지에서 템플릿 옆에 있는 생략 부호(...)를 클릭하고 **[!UICONTROL 삭제]**&#x200B;를 선택합니다.

  ![복제 및 삭제 작업에 액세스하려면 ...을 클릭합니다](./assets/templates-list-more-menu.png){width="500"}

이 작업을 수행하면 확인 대화 상자가 열립니다. **[!UICONTROL 취소]**&#x200B;를 클릭하여 프로세스를 중단하거나 **[!UICONTROL 삭제]**&#x200B;를 클릭하여 제거를 확인할 수 있습니다.

## 일괄 작업 수행

랜딩 페이지 템플릿 목록 페이지에서 왼쪽의 확인란을 선택하여 한 번에 여러 템플릿을 선택합니다. 여러 템플릿을 선택하면 하단에 배너가 나타납니다.

![배너에 선택한 템플릿 수와 삭제 아이콘이 표시됩니다](./assets/templates-multi-select-banner.png){width="600"}

**[!UICONTROL 삭제]** — 한 번에 최대 20개의 템플릿을 삭제할 수 있습니다. 확인 대화 상자를 통해 작업을 중단하거나 템플릿 제거를 확인할 수 있습니다.

## 저장된 템플릿에서 랜딩 페이지 작성

_랜딩 페이지 만들기_ 페이지에서 _디자인 템플릿 선택_ 섹션을 사용하여 템플릿에서 콘텐츠를 빌드합니다.

생성된 랜딩 페이지 템플릿 중 하나를 사용하여 콘텐츠 작성을 시작하려면 다음 단계를 수행하십시오.

1. _콘텐츠 편집_ 페이지에서 시각적 디자인 공간에 액세스합니다.

   _랜딩 페이지 만들기_ 페이지에서 기본적으로 _샘플 템플릿_ 탭이 선택됩니다.

1. 사용자 지정 랜딩 페이지 템플릿을 사용하려면 **[!UICONTROL 저장된 템플릿]** 탭을 선택하십시오.

   이 탭에는 샌드박스에서 만든 모든 랜딩 페이지 템플릿 목록이 표시됩니다. _이름별_, _마지막으로 수정한 날짜_ 및 _마지막으로 만든 날짜_&#x200B;로 정렬할 수 있습니다.

1. 목록에서 원하는 템플릿을 선택합니다.

   선택 후 템플릿의 미리보기가 표시됩니다. 미리보기 모드에서 오른쪽 및 왼쪽 화살표를 사용하여 한 범주(선택에 따라 샘플 또는 저장됨)의 모든 템플릿 사이를 이동할 수 있습니다.

1. 오른쪽 상단의 **[!UICONTROL 이 서식 파일 사용]**&#x200B;을 클릭합니다.

1. 시각적 콘텐츠 디자이너에서 필요에 따라 콘텐츠를 편집합니다.
