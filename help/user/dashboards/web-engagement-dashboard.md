# 웹 참여 대시보드

웹 참여 대시보드는 웹 방문자가 주요 콘텐츠와 상호 작용하는 방법을 보여 줍니다. 참여 트렌드를 이해하는 데 도움이 되도록 계정 업계 및 지역에 걸쳐 데이터를 세그먼트화합니다. 이 대시보드를 사용하여 콘텐츠 전략 및 계정 타겟팅을 알리는 웹 행동 패턴을 표시하여 전략적 의사 결정을 지원합니다.

_웹 참여 대시보드_&#x200B;에 액세스하려면 왼쪽 탐색에서 **[!UICONTROL 대시보드]** 항목을 선택하십시오. 그런 다음 페이지 상단의 **[!UICONTROL 웹 참여]** 탭을 선택합니다.

![웹 참여 대시보드](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## 데이터 필터링

다음 특성을 사용하여 표시된 데이터를 필터링하려면 왼쪽 상단의 _필터_( ![필터 아이콘](../assets/do-not-localize/icon-filter.svg)) 아이콘을 클릭하십시오.

* **[!UICONTROL 계정 영역]** - 계정과 연결된 하나 이상의 선택된 지리적 영역별로 데이터를 필터링합니다.
* **[!UICONTROL 계정 업종]** - 계정과 연결된 하나 이상의 선택한 산업 분류별로 데이터를 필터링합니다.
* **[!UICONTROL 날짜 범위]** - 선택한 날짜 범위별로 데이터를 필터링합니다. 기본 범위는 현재 날짜입니다.

![특성별로 표시된 데이터를 필터링합니다](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

데이터를 필터링하는 데 사용할 각 특성의 값을 선택하고 **[!UICONTROL 적용]**&#x200B;을 클릭합니다.

## [!UICONTROL 상위 페이지 보기 수] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="상위 페이지 조회수"
>abstract="웹 사이트에서 가장 자주 보는 페이지로, 가장 많은 트래픽을 유도하는 콘텐츠를 식별하는 데 도움이 됩니다."

이 표에는 가장 많이 본 상위 10개 웹 페이지가 표시되어 어떤 콘텐츠가 방문자에게 가장 큰 반향을 일으키는지 식별하는 데 도움이 됩니다. 이 데이터에는 다음이 포함됩니다.

| 열 | 설명 |
| ------ | ----------- |
| 페이지 이름 | 웹 페이지의 이름 또는 제목입니다. |
| 총 보기 수 | 페이지를 본 총 횟수입니다. |
| 알려진 방문자 수 (%) | 알려진(확인된) 방문자에 기인한 페이지 보기 횟수의 비율입니다. |
| 알 수 없는 방문자 수 (%) | 알 수 없는(익명) 방문자로 인한 페이지 보기의 비율입니다. |

![상위 페이지 보기 횟수 테이블](./assets/web-engagement-dashboard-top-page-views.png){width="650" zoomable="yes"}

## [!UICONTROL 계정 영역별 페이지 보기] {#page-views-by-region}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_page_views_by_region"
>title="계정 지역별 페이지 조회수"
>abstract="연결된 계정의 지리적 영역으로 세그먼트화된 웹 방문자 배포입니다."

이 시각화는 계정 영역으로 세그먼트화된 방문자 수를 표시합니다. 웹 트래픽이 서로 다른 지리적 영역에 걸쳐 어떻게 달라지는지 보여 주므로, 콘텐츠 및 캠페인을 지역 대상자에게 맞춤화할 수 있습니다. 다음을 포함한 세부 정보를 보려면 차트의 막대 위로 마우스를 가져갑니다.

* 계정 영역 이름
* 페이지 보기 수

![계정 영역별 페이지 보기](./assets/web-engagement-dashboard-page-views-by-account-region.png){width="500" zoomable="yes"}

>[!CONTEXTUALHELP]

이 시각화는 계정 산업별로 세그먼트화된 방문자 수를 표시합니다. 이 차트를 사용하여 웹 트래픽이 업종에 따라 어떻게 다른지 이해함으로써 업종별 콘텐츠 전략을 개발할 수 있습니다. 다음을 포함한 세부 정보를 보려면 차트의 막대 위로 마우스를 가져갑니다.

* 계정 업계 이름
* 페이지 보기 수

![계정 산업별 페이지 보기](./assets/web-engagement-dashboard-page-views-by-account-industry.png){width="500" zoomable="yes"}

## 데이터 활용

데이터에 참여하려면 각 차트의 오른쪽 상단에 있는 _자세히_(**...**) 메뉴를 사용하고 **[!UICONTROL 자세히 보기]**&#x200B;를 선택하여 확장된 데이터와 통찰력을 봅니다.

표시되는 팝업에는 데이터 분류를 보여 주는 차트와 테이블이 포함됩니다.

데이터를 다운로드하려면 데이터 테이블 오른쪽 상단의 **[!UICONTROL CSV 다운로드]**&#x200B;를 클릭합니다.

![상위 페이지 보기 테이블 추가 팝업 보기 - CSV 다운로드 클릭](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}
