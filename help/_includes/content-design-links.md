# 콘텐츠 작성 - 링크 추적

링크 추적 패널은 콘텐츠에서 각 URL을 쉽게 찾을 수 있는 중앙 집중식 보기를 제공합니다. 왼쪽의 _[!UICONTROL 링크]_ 아이콘(![링크 표시 아이콘](../assets/do-not-localize/icon-links.svg))을 클릭하여 콘텐츠에 연결된 모든 URL을 표시합니다. 텍스트, 버튼 또는 이미지 구성 요소 내의 링크입니다.

나열된 콘텐츠 링크에 대한 추적을 정의하려면 _편집_( ![편집 아이콘](../user/assets/do-not-localize/icon-edit.svg) ) 아이콘을 클릭하고 **[!UICONTROL 추적 유형]** 또는 **[!UICONTROL 레이블]**&#x200B;을 수정하십시오.
